ProgramControl connects the command line interface to FileHandler, requesting file
contents and returning readable text to the user. It never accesses files directly.

UML:

Class ProgramControl
~~~~~~~~~~~~~~~~~~~~
+ProgramControl() # Constructor that initializes FileHandler and builds the available file list dynamically from the data folder.
+getAvailableFiles():ArrayList<String> # Returns the list of filenames found in the data folder.
+printFileList():void # Prints a numbered list of available files to the console, zero-padded to match the assignment format (e.g. "01 file.txt").
+getFileContentsByNumber(int fileNumber):String # Returns the full text content of the file at the given 1-based index.
Returns an error string if the number is out of range or the file cannot be read.
